Tests en consoles

Le développement s'est entièrement réalisé en console. Il est possible d'utiliser la classe créée comme suit:

Création d'un objet de type float binaire

let float = new FloatingType('123.12');

Paramètres possibles:

  • Int
  • Chaine de caractère représentant un nombre
  • Infinity
  • NaN

Opérations disponibles

Plusieurs opérations sont disponibles sur cette classe, à noter qu'un objet est immutable.

let a = new FloatingType('123.12');
let b = new FloatingType('-12');
let result = a.add(b);
result.toString();

Liste des fonctions disponibles:

  • toString()
  • add(n)
  • sub(n)
  • mult(n)
  • divBy(n)
  • isNaN()
  • isZero()
  • isInfinity()

Une fonction static est également disponible:

let a = FloatingType.oneBy('123.12');
