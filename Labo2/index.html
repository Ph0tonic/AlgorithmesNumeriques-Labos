Contextualisation

Dans le cadre du deuxième laboratoire d'algorithme numérique (Haute-École Arc : Module 2231.3), il nous a été demander de réaliser un algorithme JavaScript pour pouvoir trouvé les racines des fonctions suivante :

  • sin(x) - x / 13 = 0
  • x / (1-X^2) = 0

Nous devons pouvoir trouvé toutes les solutions dans l'intervale (ademis fermé,fermé) : [100,100].

Les algorithmes et les contrainte suivante nous ont été imposé :

  • Dichotomie : Calculer l'erreur de l'algorithme
  • Pente (Newton) : Si le X0 choisi ne permet pas de converger vers une valeur, l'algorithme nous demandera d'augementer ou de baisser la valeur du X0
  • Point fixe : Ajuster la valeur de lamba dynamiquement pour trouver une solution

Structure du code

Afin de pouvoir intégrer n'importe quel algorithme à notre solution. Nous avons opté sur un développement orienté objet utilisant l'héritage. Nous avons donc deux class principal : Plot et Algorithm (classe abstraite). Puis pour chacun des algorithmes nous avons créé une classe dérivée de la classe Algorithm. Chacune des classes dérivés doit ensuite implémenter la fonction draw() et nextStep(). La class Plot, comme son nom l'indique, permet de gérer l'affichage du graphique mais aussi de gérer les 'pas' de l'algorithme.

Avec cette structure de code, il nous est très facile d'ajouter d'autre algorithme et également de modifier leurs comportements.

Methode de la tangante, aussi appelé méthode de la pente ou méthode de Newton (classe Tangent)

Pour cette classe, nous n’avons eu que besoin "d'un point de départ" duquel l'algorithme va commencer à essayer de converger vers la racine.

Nous avions normalement à implémenter une fonction d'approximation de la dérivée d'une fonction en un point (droite tangente). Elle est basé sur la définition de la dérivé, où nous choisissons un "h" (aussi appelé delta ou k) très faible puis selon le principe des approximations linéaire trouvé la valeur de la dérivé.

Mais alliant très bien compris comment implémenté une tel fonction ajouté au fait que nous trouvé une bibliothèque qui permet de trouvé la dérivé analytique d'une fonction. Nous avons choisi simplement d'évaluer la dérivée analytique au lieu d'utiliser une telle approximation.
